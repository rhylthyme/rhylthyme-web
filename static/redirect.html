Rhylthyme D3 Visualizer with PatternFly

Quick Start

To use the visualizer, you need to specify a program file. Use one of the examples below or create your own.

Example Programs

  • Breakfast Schedule
  • Select Custom File

Note: The web server visualization is deprecated. Use the new integrated web visualizer for better performance and features.

The server-based visualization has been replaced with a static HTML generator that creates self-contained visualization files. Use one of the following methods:

Command Line

Using CLI:

rhylthyme visualize /path/to/your/program.json

Using Python module:

python -m rhylthyme.web_visualizer /path/to/your/program.json
